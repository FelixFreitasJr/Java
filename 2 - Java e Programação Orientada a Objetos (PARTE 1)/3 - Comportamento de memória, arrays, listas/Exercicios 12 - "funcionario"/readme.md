<h3>
Fazer um programa para ler um número inteiro N e depois os dados (id, nome e salario) de N funcionarios. Não deve haver repetição de id.<br>
<br>
Em seguida, efetuar o aumento de X por cento no salario de um determinado funcionario. Para isso, o propgrama deve ler um id e o valor X. se o id informado não existir, mostrar uma mensagem e abortar a operação. Ao final, mostrar a listagem atualizada dos funcionarios, conforme exemplos.<br>
<br>
Lembre-se de aplicar a tecnica de encapsulamento para nao permitir que o salario possa ser mudado livremente. Um salario so pode ser aumentado com base em uma operação de aumento por porcentagem dada. 
</h3>
